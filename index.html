Pusat Bantuan & Panduan

Temukan solusi cepat dan dokumentasi cara kerja Minestack AI Architect.

Bagaimana cara mengganti API Key dan URL AI?

Anda dapat pergi ke tab API Keys di sidebar kiri (ikon kunci). Di sana, masukkan Base URL, API Key, dan ID model (contoh: combo1) yang Anda miliki lalu klik "Simpan Konfigurasi". Koneksi baru Anda akan langsung digunakan secara real-time.

Bagaimana cara mendownload seluruh dokumen spesifikasi?

Saat Anda membuka detail proyek di Workspace, klik tombol Download ZIP yang terletak di sudut kanan atas toolbar. Seluruh 6 dokumen markdown akan dikemas dalam format .zip dan diunduh langsung ke komputer Anda.

Format diagram apa yang didukung di visualizer?

Minestack AI Architect menggunakan parser Mermaid.js yang mendukung diagram berbasis teks. Anda dapat menulis diagram relasi database (ERD), diagram alir sistem, dan diagram arsitektur tingkat tinggi langsung menggunakan tag ```mermaid ... ``` di mode edit teks mentah.
